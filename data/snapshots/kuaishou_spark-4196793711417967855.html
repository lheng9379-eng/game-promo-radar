# 达人开放平台接入指引（星火计划小程序任务）

更新时间：2025-10-31 22:41:30

# 一、平台介绍及入口

「达人开放平台」为快手小程序官方的撮合平台，是基于小程序设立的生态营销平台，为小程序开发者提供海量达人资源。小程序开发者可通过在平台内发布任务的形式，号召更多达人通过短视频/直播的形式对小程序内容进行推广，为小程序带来更多的用户、点击和收入。

「达人开放平台」主要基于小程序的生态营销平台，有营销诉求的开发者可通过PC端达人开放平台 (opens new window)发布推广任务，平台将任务聚合在快手APP端「星火计划」功能中，邀请达人进行接单，完成开发者-达人的撮合，帮助达人在快手实现端内创作变现，帮助开发者实现流量获取与商业变现。

开发者侧界面及入口示意

达人开放平台界面入口：快手小程序开发者平台-广告中心-广告主-达人推广

用户侧界面及入口示意

星火计划界面入口1：快手搜索"星火计划"入口2：快手侧边栏-创作者服务-星火计划入口3：快手侧边栏-小程序-星火计划

# 二、开放规则

目前快手达人开放平台（星火计划）面向全量快手小程序开发者开放，已覆盖长视频、小说、娱乐测试及工具、运营商服务等重点行业，接受设定广告分成、收入分成、线索转化三种类型的任务。同时，目前已支持按支付金额分成计费（cps）、按广告曝光数计费（cpm）、按线索留资转化计费等计费方式。

# 四、接入指引

# 4.1 平台入驻及协议签署

协议地址：达人开放平台-协议签署-小程序推广协议 (opens new window)

完成协议签署：使用小程序管理员账号(账号查看 (opens new window))登陆达人开放平台，点击协议签署中小程序推广协议，开发者需仔细阅读平台提供的授权材料后完成签署。

# 4.2 账户充值（仅线索转化及收入分成比例>27%的任务需要充值）

若开发者需要创建分成比例大于27%的收入分成任务或线索转化任务，需要在平台【账户管理】中完成账户充值。充值完毕后，开发者可在创建任务时使用账户余额作为任务预算。创建预充值任务，单个任务金额不得低于100元。账户充值界面示意：

2、注意事项：

请仔细核对收款账户信息和打款主体无误后再进行汇款，收款方账号为【1109288877104077710407005】。

# 4.9 结算

# 4.9.1 开发者视角结算

分成类型结算说明
收入分成

任务发布后，快手达人A发布了视频并挂载相应的小程序。用户通过点击视频左下角链接进入小程序产生支付，该支付金额会归因到达人A的视频。当T+n后小程序开发者侧发起结算，平台会获得结算金额，并按照「视频的结算金额*达人佣金率」作为创作者收益。

当分成比例小于等于28%时，平台自动分账，无需充值

 

当分成比例大于28%时，需要充值并绑定任务预算，绑定的任务预算会冻结至任务结束

 

退款：

退款场景说明，开发者发起结算后用户退款不影响达人结算

广告分成

任务发布后，快手达人A发布了视频并挂载相应的小程序。用户通过点击视频左下角链接进入小程序产生支付，该支付金额会归因到达人A的视频。当T+n后小程序开发者侧发起结算，平台会获得结算金额，并按照「视频的结算金额*达人佣金率」作为创作者收益。

举例：任务分成比例为80%，假如一个达人发布了一条视频，通过计算该视频内，用户进入小程序看广告的效果，预计给开发者带来的广告收入A，开发者设置的分成比例是80%，则达人收入是A*80%*（1-10%）。（10%为平台向达人收取的服务费）

线索转化

任务发布后，达人发布了视频并挂载相应的小程序。用户通过点击视频左下角链接进入小程序产生线索留资/激活/充值，该线索需要开发者回传至平台，平台按照「开发者回传的线索数*线索价格」作为创作者收益。

举例：任务设置一个线索70元，假如一个达人发布了一条视频，通过统计该视频内带来的线索留资/激活/充值情况，有A条有效线索，达人收入是A*70元*（1-10%）。（10%为平台向达人收取的服务费）

# 4.9.2 达人视角结算

结算主体

目前星火平台入驻的达人分为三种类型，分别为自然入驻达人、MCN机构达人及服务商达人。目前，自然入驻达人通过任务获得的收益会由平台直接结算给达人（扣除10%的手续费），而MCN机构达人、服务商达人，则由平台结算给达人绑定的MCN机构或撮合服务商，由服务商给达人进行自行结算。


 

结算规则

当前平台的审核环节采用的是「先发作品后审核」，在给作者下发任务收益前，平台会审核该视频是否满足任务要求。如果满足则给作者下发任务收益，否则不予发放。同时，如达人在经营过程中存在作弊、刷单等恶劣行为，平台也同时会对于该达人当前结算周期的收入不予结算。


 

达人领取收益流程作者发布作品并挂载了商单任务后，每月11日～25日之间产生的收益会在下月1日下发到达人钱包。上月26日～本月10日内产生的收益会在本月16日下发到达人钱包。作者可在星火页面的钱包中提现。
